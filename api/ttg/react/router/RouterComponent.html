RouterComponent 

trait RouterComponent[Info, To] extends AnyRef

Router component for your react tree. You can nest as many RouterComponents as you want just recognize that RouterComponent is really designed to show content based on the current history state, including the ability to swich itself off rendering to a null react node. So think of this component as a switch based on document location where the document location is a global variable.
